Funciones Matemáticas: Definición

  • En matemáticas, para las funciones definidas como \(f : A \rightarrow B\), el conjunto \(A\) se denomina dominio y el conjunto \(B\) se denomina rango.
  • A partir de esta definición se construye el encabezado o firma de las funciones de programación.
## Funciones en Programación

Funciones en Programación: Definición

Una función, desde el punto de vista de la programación, se define como un proceso que recibe valores de entrada (llamados argumentos) y el retorna un valor resultado.

De la definición matemática: $$f : A \rightarrow B$$

se tiene que \(f\) corresponde al nombre de la función, el conjunto \(A\) corresponde al tipo de los argumentos de dicha función y el conjunto \(B\) que es el rango corresponderá al valor de retorno de dicha función.

## Funciones en Programación: C++ ```cpp #include <iostream> using namespace std; // Espacio para declarar funciones bool esPar(int x) { return x % 2 == 0; } int main() { return 0; } ```
## Funciones en Programación: Python ```python # Espacio para declarar funciones def esPar(x): return x % 2 == 0 ```

Funciones en Programación: Ejemplo

Se definirá una función que eleve un número al cubo. Para expresar una función que calculé esta operación, en primera instancia se construye la expresión \(f : \mathbb{R} \rightarrow \mathbb{R}\) que define que la función tiene como entrada (dominio) un número real y como salida (rango) un número real. La declaración de la función quedará de la siguiente forma: $$f: \mathbb{R} \rightarrow \mathbb{R}$$ $$f: x 􏰀\rightarrow x^2$$

## Funciones con varios argumentos

Funciones con varios argumentos

Las funciones están definidas de la forma \(f: A \rightarrow B\), siendo \(A\) y \(B\) conjuntos. Esta definición nos permite utilizar productos cartesianos generalizados como dominio en la declaración de las funciones.

I.e., \(A = C \times D\) por lo tanto \(f: C \times D \rightarrow B\).

Funciones con varios argumentos: Ejemplo

La función que calcula el área de un rectángulo es una función que tiene como parámetros de entrada el ancho y el largo del rectángulo que son valores de tipo real y retorna como salida el área del rectángulo. Lo que puede escribirse como: $$areaRectangulo: \mathbb{R} \times \mathbb{R} \rightarrow \mathbb{R}.$$

Funciones con varios argumentos: Ejemplo

Para el calculo del área de un rectángulo es necesario conocer el largo y el ancho del rectángulo, a partir de los cuales el área del rectángulo está dada por la expresión \(A_r = l * a\), donde las variables están definidas así:

  • \(l\) := Largo del rectángulo
  • \(a\) := Ancho del rectángulo
  • \(A_r\) := Área del rectángulo de largo \(l\) y ancho \(a\)

Funciones con varios argumentos: Ejemplo

Y la función matemática queda definida de la siguiente forma: $$areaRectangulo: \mathbb{R} \times \mathbb{R} \rightarrow \mathbb{R}$$ $$areaRectangulo: (l, a) \rightarrow l * a$$

## Funciones con varios argumentos: Ejemplo En este caso lo único que cambia es la firma de la función, así:
## Funciones con varios argumentos: Ejemplo en C++ ```cpp #include <iostream> using namespace std; float areaRectangulo(float l, float a) { return l * a; } double areaRectangulo(double l, double a) { return l * a; } int main() { float l = 0; float a = 0; float Ar = 0; cout << "Ingrese el largo del rectangulo: \t"; cin >> l; cout << "Ingrese el ancho del rectangulo: \t"; cin >> a; Ar = areaRectangulo(l, a); cout << "El area del rectangulo de largo " << l; cout << " y ancho " << a; cout << " es: " << Ar << endl; return 0; } ```
## Funciones con varios argumentos: Ejemplo en Python ```python def areaRectangulo(l, a): return l * a l = float(input("Ingrese el largo del rectangulo: \t")) a = float(input("Ingrese el ancho del rectangulo: \t")) Ar = areaRectangulo(l, a) print("El area del rectangulo de largo " + str(l), end="") print(" y ancho " + str(a), end="") print(" es: " + str(Ar)) ```
## Funciones con varios argumentos: Ejemplo Note que los conjuntos que hacen parte del producto cartesiano del dominio pueden ser de diferentes tipos:
## Funciones con varios argumentos: Ejemplo en C++ ```cpp #include <iostream> using namespace std; char sumaCaracteres(char c, int n) { return c + n; } int main() { char c = 0; int n = 0; char newC = 0; cout << "Ingrese un caracter: \t"; cin >> c; cout << "Ingrese numero: \t"; cin >> n; newC = sumaCaracteres(c, n); cout << "La suma del caracter: " << c; cout << " con el numero: " << n; cout << " da como resultado: " << newC << endl; return 0; } ```
## Funciones con varios argumentos: Ejemplo en Python ```python def sumaCaracteres(c, n): return ord(c) + n c = input("Ingrese un caracter: \t") n = int(input("Ingrese numero: \t")) newC = chr(sumaCaracteres(c, n)) print("La suma del caracter: " + str(c), end="") print(" con el numero: " + str(n), end="") print(" da como resultado: " + newC) ```

## Variables Globales y Locales
## Variables Globales: Ejemplo en C++ ```cpp #include <iostream> using namespace std; int a = 2; int multiplicar(int b) { return a * b; } int main() { multiplicar(2); a++; cout << a << endl; return 0; } ```
## Variables Globales: Ejemplo en Python ```python a = 2 def multiplicar(b): return a * b multiplicar(2) a+=1 print(a) ```
## Variables Locales: Definición Variables que son definidas dentro de cada función y/o procedimiento, y que solo se pueden usar en la función y/o procedimientos en la que son declaradas.
## Variables Locales: Ejemplo en C++ ```cpp #include <iostream> using namespace std; int sumar(int a) { return ++a; } int restar(int a) { return --a; } int main() { int s = 0; int r = 0; s = sumar(3); r = restar(80); cout << s << endl; cout << r << endl; return 0; } ```
## Variables Locales: Ejemplo en Python ```python def sumar(a): return a + 1 def restar(a): return a - 1 s = sumar(3) r = restar(80) print(s) print(r) ```
## Variables Globales y Locales: Ejemplo en C++ ```cpp #include <iostream> using namespace std; double x = 1.67; int cuadrado(int x) { return x * x; } int main() { int c1 = 0; c1 = cuadrado(3); double c2 = x * x; cout << c1 << endl; cout << c2 << endl; return 0; } ```
## Variables Globales y Locales: Ejemplo en Python ```python x = 1.67 def cuadrado(x): return x * x c1 = cuadrado(3) c2 = x * x print(c1) print(c2) ```
## Variables Globales y Locales: Recomendaciones Una buena técnica de programación es evitar o usar la menor cantidad de variables globales, de tal forma que las funciones y/o procedimientos que se creen no dependan de elementos externos, en este caso las variables globales, para realizar su proceso. El no usar variables globales dentro de una función y/o procedimiento garantiza su fácil depuración y seguimiento.
## Argumentos por Valor y Referencia
## Argumentos por Valor Los argumentos convencionales son por valor, es decir a la función o procedimiento se le envía un valor que almacena en la variable correspondiente al argumento, la cual es local, de manera que su modificación no tiene efecto en el resto del programa.
## Argumentos por Referencia Sí una función/procedimiento tiene un argumento por referencia, este no recibe un valor, sino una referencia a una variable, es decir la misma variable (posición en memoria y valor) que envía el algoritmo que hace el llamado al procedimiento con un alias (el nombre de la variable del argumento que se recibe por referencia), por lo tanto cualquier modificación del argumento tiene efectos en el algoritmo que realizó el llamado de la función/procedimiento.
